Which Layout?

Layout 1 (Position Columns): Best for "do I have a SS yet?" — scan each position column to see who's left. Classic cheat sheet feel.
Layout 2 (Tier x Position Grid): Best for "what tier am I drafting in?" — see which positions still have elite/solid/mid options at a glance.
Layout 3 (Category Cheat Sheet): Best for H2H categories — organized by what wins you matchups. "I need SB" → scan the SB column. Shows the same player in multiple columns if they contribute to multiple categories.
